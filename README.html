jQueryAnimatedChart

Animated jQuery pie chart built on top of jQuery Knob. Chart has a colored circle label that has a title, a tagline and a percentage value. Chart can be customized through SCSS.

Example

``` html

Chart Title 1 tagline %

```

Compatibility

Tested on Firefox 33.1, Chrome 38.0, Safari 8.0, Mobile Safari 7.0, Android Chrome 39.0.
